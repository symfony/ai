Symfony AI Documentation
========================

Symfony AI is a comprehensive set of components that integrate artificial intelligence capabilities into PHP applications. 
It provides a unified interface for working with various AI models and platforms while maintaining the simplicity and 
elegance that Symfony developers expect.
